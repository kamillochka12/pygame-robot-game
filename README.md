# Робот-сборщик

Учебная 2D-игра на `Python` и `pygame`. Игрок управляет роботом-сборщиком, ловит полезные детали, избегает вредных предметов и старается набрать как можно больше очков.

## Описание игры

В игре есть главное меню, выбор сложности и два режима:

- одиночная игра;
- игра на двух игроков.

Во время игры сверху падают полезные и вредные предметы. За полезные детали начисляются очки, а за столкновение с вредными предметами снимаются жизни. Когда жизни заканчиваются, появляется экран завершения игры с итоговым результатом.

## Возможности

- главное меню;
- выбор уровня сложности;
- режим на одного игрока;
- режим на двух игроков;
- подсчёт очков;
- система жизней;
- экран завершения игры.

## Управление

### В игре

- Игрок 1: `←` и `→`
- Игрок 2: `A` и `D`

### В меню

- `↑` и `↓` — выбор пункта;
- `←` и `→` — изменение режима или сложности;
- `Enter` — запуск игры или подтверждение;
- `Esc` — выход.

## Использованные технологии

- `Python`
- `pygame`

## Запуск проекта

1. Установить библиотеку `pygame`:

```bash
pip install pygame
```

2. Запустить файл игры:

```bash
python main.py
```

## Структура проекта

- `main.py` — основной код игры;
- `assets/` — изображения для фона, роботов и предметов;
- `README.md` — описание проекта.
